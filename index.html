Emulators. Emulators everywhere.

Virt.js allows you to create embeddable emulators on a variety of architectures. More than that, it exposes conventioned public API so you can easily plug them into your own projects.

Open Source

Virt.js is released under the GPL license, giving it free of charge to every open-source project that you could think of. The world is yours!

Modular

Virt.js aims to give you full leverage on the running emulator. It allows you to write your own devices should you need to, and directly wire them into the core.

Fast

Thanks to an advanced JIT process, Virt.js is able to achieve incredible speed, and some engines can even run flawless on latest mobile browsers.

// By the way, this exact code is used for launching the emulator above :)import { WebGLScreen }    from 'virtjs/devices/screens/WebGLScreen';import { KeyboardInput }  from 'virtjs/devices/inputs/KeyboardInput';import { fetch }          from 'virtjs/utilities/fetch';import { Engine, inputs } from 'virtjs-gbjit/Engine';var screen = new WebGLScreen( { element : document.querySelector( '#screen' ) } );var input = new KeyboardInput( { inputs : inputs } );var engine = new Engine( { devices : {    screen : screen,    input : input} } );var rom = fetch( '/roms/2048-sanqui.gb' );engine.loadArrayBuffer( rom );
